# Security Policy

If you discover a security vulnerability within Suffete, please send an e-mail to our Security Team via [security@carthage.software](mailto:security@carthage.software).

Please withhold public disclosure until after we have addressed the vulnerability.

There are no hard and fast rules to determine if a bug is worth reporting as a security issue. Suffete is a library: it does not run code, accept network input, or hold credentials, so most defects are functional bugs that belong in the public issue tracker. When in doubt, err on the side of contacting us privately first.
